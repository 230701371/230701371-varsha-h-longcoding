Parking Lot Management System

Overview
This project is a simple and efficient Parking Lot Management System developed using Object-Oriented Programming concepts.  
It manages vehicle parking, tracks slot occupancy, and calculates parking charges based on duration.

Features
Automatic parking slot allocation
Real-time occupancy tracking
Time-based billing system
Efficient slot reuse after vehicle exit
Clean OOP-based design

Tech Stack
Language: Python  (change to Java if needed)
Concepts: OOP (Classes, Objects, Methods)
